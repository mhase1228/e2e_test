## ログイン（TC-LOGIN）

### TC-LOGIN-01

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| テストID   | TC-LOGIN-01                    |
| テスト名   | 正常ログイン（プレミアム会員） |
| 対象ページ | /ja/login.html                 |
| テスト種別 | 正常系                         |
| 優先度     | High                           |
| 前提条件   | 未ログイン状態                 |

**手順**
1. `/ja/login.html` を開く
2. メールアドレス欄に `ichiro@example.com` を入力する
3. パスワード欄に `password` を入力する
4. 「ログイン」ボタンをクリックする

**期待結果**
- `/ja/mypage.html` に遷移する
- ナビゲーションに「マイページ」と「ログアウト」が表示される
- ナビゲーションから「会員登録」と「ログイン」ボタンが消える

## 会員登録（TC-SIGNUP）

### TC-SIGNUP-01

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| テストID   | TC-SIGNUP-01                                       |
| テスト名   | 正常な会員登録（必須項目のみ）                     |
| 対象ページ | /ja/signup.html                                    |
| テスト種別 | 正常系                                             |
| 優先度     | High                                               |
| 前提条件   | 未ログイン状態、登録済みでないメールアドレスを使用 |

**手順**
1. `/ja/signup.html` を開く
2. メールアドレス欄に未登録のメールアドレスを入力する（例: `newuser@example.com`）
3. パスワード欄に `password123` を入力する
4. パスワード（確認）欄に `password123` を入力する
5. 氏名欄に `テスト太郎` を入力する
6. 会員ランクは「プレミアム会員」のままにする
7. 「登録」ボタンをクリックする

**期待結果**
- マイページ（`/ja/mypage.html`）に遷移する
- 登録したメールアドレス・氏名が表示される
- 会員ランクが「プレミアム会員」と表示される
